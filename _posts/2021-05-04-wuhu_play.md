# 芜湖行

## 路线

我们是从合肥站始发终到芜湖站，再做公交车到镜湖。沿途经过了不少美景，如巢湖、歪山、铃子山等，这些山上都有半山腰的人家，确实让我大开眼界。

![](/assets/img/post_img/20210504/train.png){: width = "700" .normal}

## 游览

![](/assets/img/post_img/20210504/xiaojinghu.jpg){:style="max-width: 350px" .left}经过公交车的颠簸之后，我们来到了镜湖园区，这里有大镜湖和小镜湖。在镜湖旁是安师大的赭山校区，这里应该是老校区了，由于疫情人也比较少。在湖旁的感觉就是：心旷神怡。沿着镜湖岸边的木桥移动，可以兜兜转转把一圈的景色都欣赏个遍。

在岸边有木质的小桥，小桥弯弯曲曲地绕着镜湖直通中心地，凉风习习很是愉悦。我们在一处小桥处拍了张合照。嘿嘿！然后顺着这边的路走到了安师门口，顺着北京中路，到小吃街买点吃的，中午吃了炒饭，炒饭这边可以打小菜，还有海苔煮的汤，吃完以后又去了**陈师傅生煎包**买了点，坐在旁边绿化区吃完了。准备下午的时候去[赭山公园](https://baike.baidu.com/item/赭山公园/10999698?fr=aladdin)逛一逛。

![](/assets/img/post_img/20210504/gujiuci.jpg){: width = "700" .normal}

顺着九华中路我们来到了赭山公园门口。进门直接直走，因为听说可以经过山顶看到长江，顺着中间路，走到了法治广场，看到了毛泽东同志像。本以为顺着石像后的山路可以一直通到山顶，可是盘延了许久山顶是一座电视中心，山腰处为戴安澜墓，于是我们只能灰溜溜下山了，向另外一个方向的山头出发。

![](/assets/img/post_img/20210504/maozedong.jpg){: width = "700" .normal}

![](/assets/img/post_img/20210504/zheshangongyuan.jpg){: width = "700" .normal}

走到另一座山脚下，抬头望去弯弯曲曲，而且路很陡。有石砖楼梯，但是觉得可能走起来很累。因此选择走柏油路，柏油路似乎只能到达山腰，走到一座好像是“太白亭”的亭座，开始走上山的土路。走了不大一会就到山顶了，地图上说是刘希平先生墓。但是我在山顶上却没有看到任何的长江的影子。在山上把菠萝啤一饮而尽，准备下山。下山我们选择的是走西南方向的一条石砖路，在这里还看到了一块牌坊~~（应该是吧？）~~。回去的路上买了袋茶叶，然后开始酝酿着去看长江了。

![](/assets/img/post_img/20210504/jiangchengruyu.jpg){: width = "700" .normal}

之后做好了准备后，顺着长江中路一路向南，再经过滨江路走到了芜湖大剧院广场这边。这边离远就能看到芜湖老海关，不得不说挺有历史感，在老海关前方就是长江了。没有诗句中所描述的那种波涛汹涌，看看来来往往的船只，呼吸着微冷的空气，此时心情务必愉悦。

![](/assets/img/post_img/20210504/dengta.jpg){: width = "700" .normal}

## Go home！

溜达完了就可以回去了，这次的小旅游玩的还是挺舒服的，关键看到了长江。以后有时间了肯定还是要逛一逛别的地方。最后的最后再附上一张夜晚下的蜀山。
